# TalentFlow AI 🚀

TalentFlow AI is a production-style Recruitment Management System built using the MERN Stack. It helps recruiters manage companies, job postings, candidates, and hiring workflows through a secure REST API with JWT authentication.

## Features

- 🔐 User Authentication (JWT)
- 👤 Role-based User Management
- 🏢 Company Management
- 💼 Job Management
- 📄 Candidate Applications (Coming Soon)
- 🤖 AI-powered Resume Matching (Planned)
- 📊 Recruiter Dashboard (Planned)

## Tech Stack

- MongoDB Atlas
- Express.js
- React.js
- Node.js
- Mongoose
- JWT Authentication
- bcrypt.js
- Thunder Client (API Testing)
